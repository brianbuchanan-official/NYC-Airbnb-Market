# Explaining the NYC Airbnb Market

## Overview
This project analyzes Airbnb listings across New York City to uncover pricing patterns, availability trends, and supply concentration by location and room type. The goal is to provide clear, business-focused insights using interactive Tableau visualizations suitable for non-technical audiences.

## Business Question
How do location, room type, and availability influence Airbnb pricing and supply across New York City?

## Tools Used
- Tableau Public
- Excel (light preprocessing)
- Public Airbnb dataset (Kaggle)

## Key Insights
- Airbnb prices vary significantly by location, with higher median prices concentrated in Manhattan and select Brooklyn neighborhoods.
- Entire home listings command a substantial price premium compared to private and shared rooms.
- The NYC Airbnb market is highly right-skewed, with most listings priced below $200 per night and a small number of high-priced outliers.
- Brooklyn and Manhattan account for the majority of Airbnb supply.
- Listings outside Manhattan tend to be available for more days per year, suggesting lower demand intensity or fewer short-term stays.
- Boroughs differ in room type composition, reflecting distinct market structures across New York City.
